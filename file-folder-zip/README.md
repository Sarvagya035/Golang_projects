# File-folder-zip

This program create folders, files and zip file.
This project focuses on working with the filesystem and handling ZIP compression — common tasks in backend development and CLI tools.
This projects uses Go standard library like os, filepath, zip/archive. 

```bash

### Folder structure

    parent/
    ├── child
    │   └── child.txt
    ├── empty-folder
    └── parent.txt

```
---

## ✨ Features
This program will do the following tasks

- Create folders
- Create files
- Create a zip with correct folder structure
